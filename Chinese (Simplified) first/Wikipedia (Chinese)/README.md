当访问中文维基百科时，优先切换到简体中文（大陆简体）进行浏览。

## 适配的中文语言

- 不转换（`zh`）
- 简体（`zh-Hans`）
- 繁體（`zh-Hant`）
- 香港繁體（`zh-Hant-HK`）
- 澳門繁體（`zh-Hant-MO`）
- 大马简体（`zh-Hans-MY`）
- 新加坡简体（`zh-Hans-SG`）
- 臺灣正體（`zh-Hant-TW`）
