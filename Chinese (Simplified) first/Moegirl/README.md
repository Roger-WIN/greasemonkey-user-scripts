当访问萌娘百科时，优先切换到简体中文（大陆简体）进行浏览。

## 适配的中文语言

- 关闭转换（`zh`）
- 简体（`zh-Hans`）
- 繁體（`zh-Hant`）
- 台灣繁體（`zh-TW`）
